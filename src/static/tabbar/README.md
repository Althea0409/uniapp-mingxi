# TabBar 图标说明

本目录用于存放 TabBar 图标资源。

## 所需图标

- `home.png` - 首页图标（未选中）
- `home-active.png` - 首页图标（选中状态）
- `study.png` - 学习图标（未选中）
- `study-active.png` - 学习图标（选中状态）
- `discover.png` - 发现图标（未选中）
- `discover-active.png` - 发现图标（选中状态）
- `profile.png` - 我的图标（未选中）
- `profile-active.png` - 我的图标（选中状态）

## 图标规格

- 尺寸：81px × 81px（推荐）
- 格式：PNG
- 颜色：未选中状态使用 #999999，选中状态使用 #2B46FE

## 临时方案

在开发阶段，可以：
1. 使用 iconfont 图标库
2. 使用 uni-ui 的图标组件
3. 使用在线图标生成工具

图标资源可稍后补充。
